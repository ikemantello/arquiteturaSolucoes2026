# Requisitos Não Funcionais

## Performance
A Netflix utiliza cache, balanceamento de carga e CDN para garantir reprodução fluida e reduzir tempo de carregamento.

## Escalabilidade
O sistema cresce horizontalmente conforme o aumento da demanda, utilizando infraestrutura em nuvem.

## Disponibilidade
A plataforma opera com alta disponibilidade, utilizando redundância e tolerância a falhas.

## Segurança
Os dados dos usuários são protegidos por criptografia, controle de acesso e proteção de conteúdo digital.

## Usabilidade
A interface é intuitiva, simples de navegar e adaptada para diferentes dispositivos e públicos.
